## Keybinded Programs:
 - kupfer (app launcher)
 - firefox (web browser)
 - newsboat (rss feed reader)
 - geary (email client)
 - kitty (terminal emulator)
 - flameshot and gnome-screenshot (screenshot program)
 - rofi (various custom scripts)
 - pactl (speaker volume)
 - dwmblocks (status bar)
